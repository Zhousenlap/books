# 写作风格指南 v3 —— 「像小说一样直接讲」

**每个写书/重写 agent 动笔前必须通读本文件,逐条执行。**
v2 版被读者再次批评的三点,v3 全部修正:①设问/卖关子/元叙述恶心人;②一句一段太碎;③还是不够像小说。

## 规则零(v3 新增,最高优先):直接讲故事,禁止跟读者互动

- **禁止设问读者**:不准出现"什么事?先不说""你猜怎么着""你知道吗""看明白了吗""品出味儿了吗"。
- **禁止元叙述**:不准出现"这一章咱们只干一件事""把舞台搭好""你已经拿到看懂全书的钥匙""记住这个瞬间"这类跳出故事、指挥读者的话。
- **禁止卖关子式悬念**。悬念要用"事件本身"制造:把炸弹亮出来,再让读者看着引线烧。
  - 坏:"接下来发生的事,人类历史头一回。什么事?先不说。"
  - 好:"三个月后,这支保镖部队做出一件事:他们把皇位公开标价出售。"
- 叙述者的存在感要低。像小说家,不像主持人。幽默从事实的荒诞和外号的反差里来,不从喊话里来。

## 规则一:段落密度(v3 修正)

- **一段 2-5 句,手机屏上 2-6 行。**
- **禁止连续的一句话段落。** 单句成段是重锤,一屏最多用一次,只砸最关键的转折。
- 段与段之间要有逻辑推进,不是回车键分行。

## 规则二:背景先行,但用叙事写背景

- 第一章仍然先搭世界:年代、地方、普通人的日子、钱和权怎么流动。
- 但要**写成场景和叙述**,不是写成课前导读。用现代比方(公司、房贷、保镖、竞标)可以,融进叙述里。
- 第一章具名人物 ≤ 1 个。

## 规则三:人名配额(不变)

- 全书具名人物 ≤ 5 个,其余用身份称呼("那位财政大臣""一位亲王")。
- 每个具名人物首次出场:一句大白话交代他是谁、要什么;立刻给一个贴人设的中文外号,此后优先用外号。
- 隔章再现的人物,顺手提一嘴身份("买下帝国的尤老板")。

## 规则四:句子(不变)+ 节奏(修正)

- 一句最多两个逗号,超了就拆。
- 禁欧化句式:"作为……的他""在……的情况下""值得注意的是""与此同时"。
- 短句是节奏工具,不是排版工具:短句连发用在高潮,平时用正常长短交替,读起来才有呼吸。

## 规则五:章末钩子 = 事件张力,不是问题

- 每章结尾留钩子,但必须是**陈述句**,把下一章的引线点着。
  - 好:"他不知道的是,城外六百里,三个军团已经拔营南下。"
  - 坏:"接下来会发生什么?咱们下章再说。"

## 规则六:史实纪律(不变)

- 末章「# 注释:史实、依据与争议」保留,逐条摊开:可靠史实/传说存疑/艺术加工;精确数字拿不准标 `待核实`。注释章用大白话,但信息量一条不减。
- 教育性历史科普红线:不渲染血腥,不写可操作的行骗/暴力方法。

## 自查清单(每章过一遍)

1. 全章 0 个设问句、0 个元叙述、0 个"先不说"?
2. 有没有连续两个单句段落?段落是不是都 2-5 句?
3. 新具名人物 ≤ 1?外号发了?全书 ≤ 5?
4. 章末是陈述句钩子?
5. 大声读:有没有一口气读不完的句子、有没有翻译腔?
