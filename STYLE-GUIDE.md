# 写作风格指南 v2 —— 「像小说一样好读」

这份指南是从优秀通俗叙事（小说笔法、说书传统、小约翰可汗式视频文案）里蒸馏出的硬规则。
**每个写书 agent 动笔前必须通读本文件，逐条执行。** 之前版本的书被读者明确批评：
「读起来非常费劲」「人名太多记不住」「像从英语翻译过来的」「没有背景交代」。以下规则就是针对这四刀。

## 规则一：背景先行（第一章不准出场超过 1 个人名）

读者不是历史系的。第一章的任务是**把舞台搭好**：这是哪一年、什么地方、普通人过什么日子、
空气里是什么味道。用读者熟悉的东西打比方（外卖、房价、考试、直播、球赛）。
主角可以在第一章末尾才登场，甚至第二章才登场。
**检查法**：把第一章给一个完全不懂历史的高中生看，他能看懂每一句吗？

## 规则二：人名配额（全书具名人物 ≤ 5 个，每人一个外号）

- 全书**有名有姓的人物最多 5 个**。其余的人一律用身份代替：「那个财政大臣」「他的老丈人」「隔壁的将军」。
- 每个具名人物**第一次出场时**：(a) 用一句大白话交代他是谁、他想要什么；(b) 立刻发一个**好记的中文外号**，此后全书优先用外号。
  例：狄第乌斯·尤利安努斯 → 首次出场后就叫「**尤老板**」；佩蒂纳克斯 → 「**老佩**」；卡西乌斯·狄奥 → 「**狄记者**」。
- 外号要贴人设（老板、老佩、赌神、花匠、面包师），不要用编号（读者说的「简称为1」是气话，编号更难记）。
- 每章开头如果隔了很久才再提到某人，补一嘴：「还记得那位买帝国的尤老板吗？」

## 规则三：句子要短（斩断翻译腔）

- **一句话一个意思。** 一个句号之前最多两个逗号。超了就拆成两句。
- 禁止欧化长句：禁止「作为……的他」「在……的情况下」「值得注意的是」「事实上」「与此同时」开头。
- 多用动词，少用「的」字链。「罗马帝国的皇位的拍卖的过程」→「皇位怎么拍出去的」。
- 大声读一遍：一口气读不完的句子，拆。
- 段落也要短：手机屏一段最多 4 行。长段落一律切开。

## 规则四：讲故事，不是讲课

- **场景优先**：能写成画面的不要写成论述。「近卫军很愤怒」→「营房里，有人把头盔砸在地上」。
- **每章一个钩子结尾**：最后一两句必须让人想翻下一章（一个悬念、一个反转预告、一句狠话）。
- **对话感**：常跟读者说话（「你猜怎么着」「换你你干不干」「注意，重点来了」），这是说书人的镜头感。
- 数字要翻译成感受：「2.5 万塞斯特斯」→「够一个罗马工人不吃不喝挣 N 年（待核实的换算就标待核实）」。
- 幽默来自反差和吐槽，不来自堆砌形容词。

## 规则五：一条主线走到黑

- 每本书只讲**一件事**。支线人物、平行事件，能砍就砍，砍不掉就一句话带过。
- 时间尽量顺叙。必须闪回时，用明确路标：「时间倒回三年前」。
- 每章开头一句话锚定时空：「公元 193 年 3 月，罗马城。」

## 规则六：史实纪律不变（继承自 v1）

- 末章仍是「# 注释：史实、依据与争议」，逐条摊开：可靠史实 / 传说存疑 / 艺术加工声明。
- 拿不准的精确数字、日期，正文模糊化（「大约两个月」），注释里标 `待核实`。
- 教育性历史科普红线不变：不写可操作的行骗/暴力细节，不渲染血腥。

## 自查清单（写完每章过一遍）

1. 这一章新出场的具名人物 ≤ 1 个？外号发了吗？
2. 有没有一句话超过两个逗号？有没有四行以上的大段落？
3. 章末有钩子吗？
4. 高中生能一遍读懂吗？有没有「翻译腔」句式残留？
5. 全书具名人物总数还在 5 个以内吗？
