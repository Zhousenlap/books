# 第二章：把医生都抓起来的国家

要讲斯大林的死，必须先讲"医生案件"。因为这桩案子，是他自己给自己挖的坑，而且挖得极深。

事情要从1948年说起。当时克里姆林宫医院有个心电图科的女医生，叫莉迪娅·季马舒克。她给一位苏联高官看病，判断是心肌梗死，主张静养；但会诊的几位权威大夫不同意她的判断。季马舒克咽不下这口气，写了封信告状，说这些大夫诊断有误。这封信当时没掀起什么风浪，被归档压下了。

一封普通的医疗纠纷告状信，就这么在档案柜里躺了四年。

要理解这封信后来的分量，得先看看1952年前后斯大林脑子里那套逻辑。他这一辈子搞政治斗争，信奉一个铁律：敌人无处不在，而且越是身边人、越是看起来忠诚的人，越可能是暗藏的敌人。从二三十年代的大清洗开始，他就是靠着"宁可错杀、不可放过"这套打法，把一切可能的威胁掐灭在萌芽里。到了晚年，这套逻辑非但没有松动，反而因为身体衰老、猜疑加剧，变得更加偏执。在这样一颗大脑里，任何一点风吹草动，都能被脑补成一场针对他的阴谋。

四年后，1952年，斯大林晚年的猜疑心正烧到最旺。有人把这封旧信重新翻了出来，送到了斯大林案头。斯大林一看，如获至宝。在他那套越来越阴暗的逻辑里，这封信证明了一件他早就"感觉到"的事：克里姆林宫的医生里，藏着一个专门谋害苏联领导人的暗杀团伙。

于是机器开动了。

1953年1月13日，苏联官方通讯社塔斯社发布一条震动全国的消息：破获一个"医生间谍破坏集团"。官方说法是，这群穿白大褂的名医，受帝国主义和境外势力指使，用治病做幌子，蓄意谋害苏联党政军的高级领导人，据说已经"害死"了好几位。被点名的医生大多是犹太人，案子还被涂上了一层浓重的反犹色彩。而那位四年前写告状信的季马舒克，摇身一变成了"祖国英雄"，1953年1月被授予列宁勋章，全国报纸拿她当警惕性的典范来歌颂。

一时间，全苏联风声鹤唳。医生成了最危险的职业，病人不敢让医生看病，医生更不敢给大人物看病——今天你给他开一片药，明天你就可能被指控是"下毒"。

这里头最黑色幽默的一环是这样的。

斯大林自己有个私人医生，叫维诺格拉多夫，看了他很多年。这位大夫医术是真好，人也算实诚。他给斯大林检查身体，发现老人血压高、健康状况不妙，就提了个专业建议：您年纪大了，应该减少工作、多多休养。

这话搁一般人身上，是关心。搁在1952年猜疑心爆棚的斯大林身上，是**大逆不道**。

在斯大林听来，"你该退下来休养"这句话，等于"你不行了，该交权了"。他勃然大怒。据后来的记载,他放出话来，要给这个医生"戴上镣铐"。于是,这位最了解他身体状况、最有资格在关键时刻救他的私人医生，进了监狱。

我们把账算一算，斯大林晚年这一通操作，等于亲手拆掉了自己周围所有的安全网：

- 最懂他身体的私人医生——抓了。
- 全国最顶尖的一批克里姆林宫名医——抓的抓，关的关。
- 剩下没被抓的医生——吓得魂不附体，谁也不敢主动往大人物身边凑。

各位，请记住这个局面。因为再过一个多月，1953年3月，当斯大林真的倒在地上、真的命悬一线的时候，苏联这个庞大帝国，居然一时找不出几个敢来、也来得了的好医生。等好不容易凑齐几个大夫赶到别墅，这些人检查斯大林的时候，手抖得连脉都快号不准——因为他们太清楚了，前面那批同行,就因为给大人物看病，正蹲在牢里。

你看，一个国家把自己的医生都抓起来了，会发生什么？

会发生这样一件事：这个国家最有权力的那个人，最后连一个能安心为他看病的医生都找不到。

而此时的斯大林还不知道，这张他亲手织了一辈子的恐惧之网，最先勒住的，会是他自己的脖子。

日历翻到了1953年2月28日。这是个星期六的夜里。孔策沃别墅要来客人了。
