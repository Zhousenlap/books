# 第五章：判他死刑的文件，由他本人签收

这位苏联外交官叫康斯坦丁·沃尔科夫，公开身份是驻伊斯坦布尔副领事，真实身份是苏联情报机关的军官。1945年8月底9月初，他找上英国领事馆，提出用情报换庇护：他掌握苏联在英国的间谍网名单，包括外交部里的两个、情报系统里的七个——其中一人，是"伦敦某反谍报科的负责人"。

各位，这个描述精确得像门牌号。外交部里的两个，对应麦克莱恩他们；"反谍报科负责人"，对应的就是菲尔比本人。沃尔科夫手里这份货，能把剑桥间谍网连根拔起。

然后，沃尔科夫提了一个要求。一个用来保命的要求，一个最终要了他命的要求。

他说：此事绝对不能用电报传回伦敦——因为苏联破译了英国的部分外交密码，电报一发，他就死定了。所有往来必须走外交邮袋，由信使亲手递送。

从保密角度看，这个要求无懈可击。它只有一个副作用：慢。外交邮袋跟着班次走，伊斯坦布尔到伦敦，一来一回就是一两个星期。沃尔科夫等于亲手给自己的救命稻草装了个延时器。

他赌的是英国人的效率。他不知道的是，效率不是问题，问题是收件人。

文件几经辗转送到伦敦军情六处，按业务归口分办——反苏谍报，第九科。于是，1945年9月的一个下午，这份要求逮捕"反谍报科负责人"的绝密文件，被端端正正放在了反谍报科负责人金·菲尔比的办公桌上。

菲尔比后来在回忆录里写过读到文件那一刻的感受，大意是：盯着纸看了很久，意识到写的就是自己。

他当晚就把消息递给了苏联联络人。

接下来是整个故事里最冷的一段操作。菲尔比主动请缨：这个案子重大，我亲自去伊斯坦布尔处理。上级批准。然后这位办案人开始了一场教科书级别的磨洋工：行程安排得七绕八绕，中途转机耽搁，赶上周末顺延——从沃尔科夫敲响领事馆大门，到菲尔比本人抵达伊斯坦布尔，前后拖了大约三个星期。这三个星期里，有多少是外交邮袋的客观延误、有多少是他故意拖延，史学家至今掰扯不清（注释里细说），但结果没有争议。

菲尔比到了伊斯坦布尔，按流程约见沃尔科夫。电话打过去，苏联领事馆那边的回应轻描淡写：没有这个人，或者，他不方便接电话。

沃尔科夫再也没有出现。

有一个流传很广的目击说法：就在那几周里，一架苏联军用飞机临时降落伊斯坦布尔，有人看见两副担架被抬上飞机，担架上的人缠满绷带。据信是沃尔科夫和他的妻子。此后他们的下落，档案里再无记载，下场可想而知，我们不展开。

菲尔比给伦敦写了结案报告，分析头头是道：可能是苏联人破译了什么，可能是沃尔科夫自己不谨慎露了马脚。总之，线索中断，建议归档。伦敦看完点点头：老金办事，还是稳。

案子就这么结了。一个想活命的人，把保命的秘密交给了秘密本身要指认的人——这句话绕，但沃尔科夫的处境比这句话还绕，而且他到死大概都不知道自己输在哪儿。

这件事还有一个后劲极大的注脚。军情六处后来复盘了无数次：如果当年沃尔科夫的材料走的是另一条线，如果收件人换一个人，冷战头二十年的谍报史就要整个重写。历史没有如果，历史只有归口管理。

而对菲尔比本人，这是他间谍生涯里最直白的一次：以前他出卖的是计划、文件、行动，这一次，是一个具体的、有名有姓的、正在求生的人。多年以后他接受采访，谈到沃尔科夫，态度平静——干我们这行的，都知道游戏规则。

记住这份平静。它后面还会出现很多次，一次比一次贵。

处理完伊斯坦布尔的"悬案"，菲尔比的仕途继续起飞。1949年，他拿到了英国谍报系统里含金量最高的驻外职位：华盛顿站，军情六处与美国中央情报局、联邦调查局的最高联络官。

用大白话说：英美两大情报机构之间所有的共享机密，都从他手里过。

莫斯科的种子轮投资，进入了爆发式回报期。
