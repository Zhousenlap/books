# 第五章：判他死刑的文件，由他本人签收

货单的主人叫康斯坦丁·沃尔科夫，公开身份是苏联驻伊斯坦布尔副领事，真实身份是苏联情报机关的军官。他在苏联这条船上看清了自己的处境，一心想跳到西方那条船上去，本书就叫他跳船人。

跳船人手里的货是真硬。他掌握苏联在英国的间谍网名单：外交部里有两个，情报系统里有七个，其中一人是"伦敦某反谍报科的负责人"。这个描述精确得像门牌号。这份货单要是顺利兑现，剑桥间谍网会被连根拔起。

然后，跳船人提了一个保命的要求，这个要求最终要了他的命。他说此事绝不能用电报传回伦敦——苏联破译了英国的部分外交密码，电报一发他就死定了。所有往来必须走外交邮袋，由信使亲手递送。

从保密角度看，这个要求无懈可击，只有一个副作用：慢。外交邮袋跟着班次走，伊斯坦布尔到伦敦一来一回就是一两个星期。跳船人等于亲手给自己的救命稻草装了个延时器。他赌的是英国人的效率。他没想到的是，效率不是问题，收件人才是问题。

文件送到伦敦军情六处，按业务归口分办。反苏谍报，归第九科。1945年9月的一个下午，这份要求捉拿"反谍报科负责人"的绝密文件，送到了反谍报科负责人金·菲尔比手里。菲尔比后来在回忆录里写过读文件那一刻的感受，大意是盯着纸看了很久，意识到写的就是自己。

他当晚就把消息递给了苏联联络人。

接下来的操作，是菲尔比一生里最冷的一段。他主动请缨：此案重大，我亲自去伊斯坦布尔处理。上级批准了。

然后这位办案人开始了教科书级别的磨洋工。行程安排得七绕八绕，中途转机耽搁，赶上周末顺延。从跳船人敲响领事馆大门，到菲尔比本人抵达伊斯坦布尔，前后拖了大约三个星期。

这三个星期里有多少是邮袋的客观延误，有多少是他故意拖延，史学家至今掰扯不清。但结果没有争议。

菲尔比到了伊斯坦布尔，按流程约见跳船人。电话打到苏联领事馆，那边回应得轻描淡写：查无此人，或者他不方便接电话。跳船人再也没有出现过。

有一个流传很广的目击说法。就在那几周里，一架苏联军用飞机临时降落伊斯坦布尔。有人看见两副担架被抬上飞机，担架上的人缠满绷带。据信是跳船人和他的妻子。此后档案里再没有这两个人的记载，下场可想而知。

菲尔比给伦敦写了结案报告，分析得头头是道：可能是苏联人破译了什么，也可能是跳船人自己露了马脚。总之线索中断，建议归档。伦敦看完点点头，觉得老基姆办事还是稳。

案子就这么结了。一个想活命的人，把保命的秘密交到了秘密要指认的那个人手里。他到死大概都不知道自己输在哪儿。

军情六处后来把这件事复盘了无数次。如果当年材料走了另一条线，或者收件人换一个人，冷战头二十年的谍报史就要整个重写。历史没有如果，历史只有归口管理。

对菲尔比本人，这是他间谍生涯里最直白的一次。以前他出卖的是计划、文件、行动。这一次是一个具体的、有名有姓的、正在求生的人。多年以后他接受采访谈到这个案子，态度平静：干这行的都知道游戏规则。这份平静往后还会出现很多次，一次比一次贵。

处理完伊斯坦布尔的"悬案"，菲尔比的仕途继续起飞。1949年，他拿到了英国谍报系统里含金量最高的驻外职位：华盛顿站，军情六处对美国中央情报局和联邦调查局的最高联络官。英美两大情报机构之间所有的共享机密，都要从他手里过。

莫斯科的种子轮投资，进入了爆发式回报期。
