# 第七章：六十六天

公元193年5月，罗马城。狠人塞的大军一天天逼近。

尤老板把能想的招，全想了一遍。

这份挣扎清单值得逐条看。你能看到买来的权力，是怎么一层层剥落的。

第一招，讲法律。

他让元老院通过决议，宣布狠人塞是"国家公敌"。从程序上，把对方定成叛贼。

这招成本最低，效果也最诚实：约等于零。

对方的军团听说自己成了"公敌"，行军速度都没变一下。

乱世里的一纸公文，分量不如士兵背囊里的一块干粮。

他还派元老院使团去半路劝阻。使团见到狠人塞，相当一部分人当场留下不走了（`待核实`：使团倒戈细节）。

第二招，派刺客。

史载他派人去军中行刺，没成。

一句话带过。反正没成，还把最后一点道义体面赔了进去。

第三招，分蛋糕。

打不过，杀不掉，尤老板干脆提议：要不咱俩一起当皇帝？

他让元老院再通过一项决议：封狠人塞为共治皇帝，请他来罗马平分大位。

细品这一幕。

两个月前，他在营墙外志在必得，全款买断皇位。

两个月后，他主动打对折，只求留一半。

狠人塞的回应是：继续行军。

对一个每天都在贬值的对手，为什么要半路接受报价？

等货自己上门，是可以免费拿的。

第四招，后世史官都不忍下笔。

据记载，走投无路的尤老板搞起了巫术和献祭，求神明保佑。（`待核实`：此说出自厌恶他的史官笔下，可能有丑化。）

一位前执政官，宦海几十年的老政客，最后的指望是念咒。

而压垮他的最后一根稻草，来自一个谁都没想到、又谁都该想到的方向。

狠人塞一路南下，几乎没打过像样的仗。他最费心思的一步棋，是攻心。

他派密使潜进罗马，给禁卫军带话。大意是：

老佩的死，我只追究当天直接动手的凶手。

其余人等，交出凶手、放下武器，就既往不咎。

停一下。还记得尤老板当初怎么赢下拍卖的吗？

靠的就是一句"我不清算"。

现在，狠人塞用同一把钥匙，开了同一把锁。

这支部队的系统里，忠诚从来不是出厂设置。趋利避害才是。

保镖们的算盘打得飞快。

跟着尤老板，就得跟野战军团拼命，大概率死。

倒向狠人塞，只要交出几个当日动手的倒霉蛋，就能接着过好日子。

这道题不用打草稿。

于是，两个月前刚把皇位卖给他的这支部队，做出了跟上次一模一样的选择：

背叛现任老板，投向出价更高的下家。

只不过这次，下家出的价码不是钱。

是命。

保镖们抓了当日动手的凶手，然后宣布：不再保护尤老板。

罗马城门，洞开。

元老院一看风向，立刻行动，麻利得像彩排过。

一次会议，三项决议：废黜尤老板，判处死刑。追尊老佩，办国葬。承认狠人塞为皇帝。

注意。两个多月前举手承认尤老板的，和今天举手判他死刑的，是同一批人。

程序永远正确。因为程序永远站在赢家那边。

士兵在皇宫里找到了尤老板。

此时他众叛亲离，卫兵散尽。偌大的宫殿，几乎只剩他一个人。

据记载，他死前留下的最后一句话，是一句茫然的质问：

"我做错了什么？我杀过谁吗？"

（`待核实`：遗言出自古代史官转述，古人笔下的临终遗言普遍有文学加工。）

公元193年6月初，尤老板被处死。过程照例只说一句。

从营墙外喊出最高价，到这一天，大约六十六天。

他花掉一生的财富和名望，买到的就是这两个月：

一份被全天下拒收的皇位。一个遗臭两千年的名声。

那句遗言乍听矫情，细想有点渗人。

因为按他自己的账本，他真没"做错什么"。

他没弑君。杀老佩的是保镖，他连知情都未必。

他没抢。是卖家主动挂牌，他按对方的规则公开出价。

程序上，元老院盖了章，头衔一个不少。

每一步单拎出来，都挑不出大毛病。

可全天下都认定他有罪。罪名不写在任何一部法典里：

他把不该标价的东西，标了价。

这场拍卖真正的破坏力，不在于换了个皇帝。罗马换皇帝多了去了。

在于它当众捅破了一层窗户纸：

原来皇位可以买。原来禁军可以卖主。原来元老院的承认，可以刀架着取得。

这些事以前也有，但都在暗地里。

从没有哪一次做得这么赤裸，这么明码标价，这么像一场正经交易。

权力这东西，最怕的不是被抢，是被看穿。

所有人都看清了它的价签，它就再也回不了神坛。

后面的历史，照着这个剧本越演越烈。

狠人塞开创了新王朝，却没能改掉军队的胃口。据说他临终传给儿子一句话：

"让士兵发财，其余人一概不用理。"（`待核实`：遗言出自史官转述。）

再往后，罗马滑进深渊：五十年里换了几十个皇帝，绝大多数死于非命。

军队立帝、索赏、弑帝、再立。流水线一样。

而那条流水线的第一次公开试运行，就是193年营墙上的这场拍卖。

最后说说赢家。

狠人塞进罗马前后干的头一件事，意味深长。

他把整支禁卫军召到城外集合。命令：不带武器，穿礼服出席。

等保镖们列好队，他的野战军团把他们团团围住。

然后新皇帝宣布：全体解散。缴械，除籍，逐出罗马百里之外。

空出来的编制，由他自己的边疆老兵顶上。

那群两度卖主、卖过一整个帝国的保镖，一夜之间失业了。

卖家交割完商品，被新老板第一个清出了场。

这大概是整个故事里，最公道的一笔交易。

故事讲完了。但先别合上书。

下一章咱们把账本摊开：哪些是钉死的史实，哪些数字对不上，古代那几位记录者各有什么毛病。

一本讲"离谱交易"的书，自己得先做个诚实的卖家。
