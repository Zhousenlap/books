# 第五章：开庭

新教皇叫斯德望六世。史书没给他留下多少个人信息，能钉死的只有两条：他的阿纳尼主教之位是老帅哥亲手祝圣的，以及他恨老帅哥。恨从哪儿来，史料没交代，后人只能从利害关系里倒推。下文叫他掘墓人，这个外号他当之无愧。

审判不是他一个人的疯狂。把这件事推上台面的，是三股拧在一起的力量。

第一股是斯波莱托的政治账。翻山王的皇冠是老帅哥加冕的。只要判定老帅哥的教皇之位从头就不合法，他生前的一切官方行为就自动作废。加冕作废，翻山王就成了假皇帝，小皇帝就是这片土地上唯一的真命天子。北边那位瘫在床上的对手，连出兵报复都难。死人嘴里，能吐出活人最想要的名分。

第二股是掘墓人自己的私账。他被老帅哥祝圣为阿纳尼主教，如今改任罗马主教，按"主教不许改嫁"的老规矩，他犯的正是老帅哥犯过的那条戒律。政敌哪天翻起这本账，他的宝座就是非法的。可要是老帅哥压根不算合法教皇，当年那次祝圣就不算数，他就从来不是什么阿纳尼主教。改嫁的罪名，自然烟消云散。判死人有罪，活人就清白了。

第三股力量藏在阴影里：二十年前整过老帅哥的那一派，残部还在罗马城里，位子还在，旧恨没凉。老帅哥当教皇的五年，这批人被压得喘不过气，如今风水轮流转。三股力量一合流，一场教会史上独一无二的审判就备齐了燃料。

会议的通知发往各教区，措辞是标准的公文腔。主教们陆续进城，进了拉特兰宫才知道要议的是什么。后来他们众口一词地说，当时想反对不敢，想走也走不了。这个说法，要等好几任教皇之后才有人敢讲出口。

897年年初，拉特兰大殿。掘墓人下令，起棺。

老帅哥已经下葬九个月。尸体被抬出墓穴，重新穿上全套教皇法衣，戴上主教冠，扶上被告席的宝座。干这些活的人心里怎么想，史书没记，只知道命令执行得一丝不苟。教皇的全副行头一件不少，远看过去，被告席上仿佛坐着一位安睡的老人。

法庭的程序同样一丝不苟，这正是整件事最瘆人的地方。主教陪审团分列两侧，书记官备好记录，公诉一方由教皇本人兼任。教会法规定被告有权答辩。死人开不了口，法庭就指派一名年轻执事站在宝座旁边，替尸体应诉。这大概是人类司法史上最难的一份辩护委托。

罪状一共三条，全是二十年前的旧账。第一条，背誓：他当年发誓永不回罗马，后来不但回了，还当了教皇。第二条，改嫁：身为波尔图主教，窃据罗马主教之位。第三条，冒行圣职：被革除教籍期间，照样以主教身份行事。三条罪状条条引经据典，起草的人显然下了功夫。

三条罪状还有个共同点：条条都是死人没法辩、活人不敢辩的。誓确实发过，教区确实换过，字面上全占得住。至于当年的誓言是被逼的、平反是前任教皇正式做出的，这些辩护理由需要有人大声讲出来。而全场敢大声讲话的，只有公诉人。

庭审的场面，后世编年史留了几笔。掘墓人情绪失控，从主位上站起来，冲着宝座上的尸体咆哮质问。流传最广的一句台词是："你当年是波尔图的主教，凭什么野心熏天，篡夺罗马宗座？"这句话是几十年后一位史家补写的，原话没人记下来，那股火气应该八九不离十。

轮到答辩，年轻执事按程序替死者说了几句。说了什么，史书没记，只记下他抖得厉害。满殿的主教鸦雀无声，没有一个人出声反对。多年之后他们解释，当时刀架在脖子上，不签字的下场清清楚楚。这个说法后来被教廷官方采纳，信不信由人。

答辩环节就这样过去了。程序要的本来就不是声音，是记录：卷宗里写明被告获得了辩护，手续齐全。中世纪的法庭未必讲理，但讲究格式。

判决毫无悬念：三条罪名全部成立。老帅哥的教皇之位被宣布为篡夺，五年任期从法律上蒸发，生前一切任命、祝圣、敕令统统作废。翻山王的皇冠，随之成了废铁。

判决当场执行。执行的仪式，教会本来是设计给活人革职用的：法衣一件一件剥下，每剥一件，宣读一句罪状。剥到最后，尸体换上一身平民粗衣。右手三根手指被砍下，那是他生前行祝福礼用的三根手指。史料对这一段的记载只有寥寥数语，本书也写到这里为止。

尸体先被扔进城外一片埋外乡人的墓地，跟无名客死者作伴。没过几天，又有人奉命把它挖出来，拖到台伯河边，抛进了河里。史料对这几步的先后说法不一，结局没有分歧：前任教皇顺着台伯河，漂向大海的方向。

在场主教在判决书上签了字，一个不落。签字名单里有一位日后也会当上教皇，他的主教职位同样是老帅哥祝圣的，这份签字对他同样有用。这条线，后面还会收回来。

掘墓人签发判决书，昭告各教区。在他看来案子办结了：政治账清了，私账平了，旧恨报了。他没算到，判决书上有一条会自己长出牙来。老帅哥当了五年教皇，祝圣和任命过的神职人员遍布意大利。而判决书白纸黑字地说，这些人的职位从来就不存在。
