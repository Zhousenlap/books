# 第三章：一份盖满印章的合同

雷斯要的东西，归根结底是一张纸。一张能让伦敦印钞厂乖乖开机的纸。

说具体点，是一份葡萄牙银行的授权合同。合同要写明，银行授权他代表一批国际投资人，去印一大批供安哥拉使用的钞票。有了这张纸，别人的印钞机就成了他的印钞机。厂子那头照单办事，根本分不清下单的是央行还是骗子。

幌子他也想好了，想得还挺高明。安哥拉这块殖民地缺钱搞发展，本就是明摆着的真事。他要编的故事，就顺着这个真事往下长。真话里掺假话，最不容易被戳穿。

他编的故事是这样的。葡萄牙银行秘密授权了一笔巨额注资，用印钞的方式，把钱悄悄送到安哥拉去搞开发。为了不惊动市场、不引来政治麻烦，整件事必须绝密。殖民地要用钱是真的，秘密行事也合情理，这故事就立住了。

这个「绝密」，是整个骗局的护身符。正因为绝密，才没有人能公开去问一句「真有这回事吗」。谁要多嘴多问，谁就是坏了国家大事的莽夫。雷斯把这一行「保密等于专业」的规矩，反手变成了自己的盾牌。制度本来用保密来防外人，如今被他拿来防查他的人。

接下来他要做的，是把这份合同做得无懈可击。他弄出一份厚厚的文件，上面盖着葡萄牙银行的印，签着总裁和几位高管的名字（签名的来源与伪造方式此处不展开）。光有这些他还嫌不够气派。

为了让文件更唬人，他张罗着让它经过公证，又设法弄来好几个国家领事馆的背书盖章（涉及几国、如何弄到待核实）。红的章、蓝的章，一层叠着一层。这份合同摊开来，庄严无比、气派十足，随手一亮就压得人喘不过气。

这里头有个反常识的道理，雷斯摸得门儿清。一份文件上的印章越多，人们反倒越不敢怀疑它。真正的授权往往简单干脆，一两个签名就了事。他偏要弄得繁复隆重，越复杂越好。

因为繁复本身就在替他说话。那些见惯了公文、迷信排场的体面人，最容易被这种阵仗唬住。他们看不懂里头的门道，只看得懂表面的气派。印章一多气派就足，气派一足人就不敢多问。雷斯赌的就是这份心理。他赌得很准。

这几个帮手里，没有一个是真去核实过的。他们只核实了一样东西，就是分给自己的钱够不够多。至于合同真假，反正有前头的人担着，有满纸的印章担着。人一旦分了赃，就自动变成了骗局的辩护人。谁拆穿它，谁就是断了自己的财路。雷斯不用一个人去骗所有人，他只要骗动第一环，剩下的人会争着替他把谎圆下去。

只是有一样，雷斯一个人办不成。他是个刚出狱的葡萄牙前科犯，身份见不得光。凭这身份，别说谈成生意，他连伦敦印钞厂那扇气派的大门都进不去。人家一查底细，立马把他轰出来。他需要体面人，需要能借上外交光的人，替他把合同送到该送的地方。

他陆续拉进了几个帮手，都是能撑场面的角色。一个是荷兰来的掮客，生意场上的老手，外表体面、门路又广（身份待核实）。一个是挂着外交头衔的葡萄牙掮客，家里有人在驻外使馆当差，能借上外交的排场和渠道（身份待核实）。还有一个德国来的合伙人，来历模糊，谈吐却很有派头（身份待核实）。这几个人凑到一处，图的都是合同上写着的那笔惊人利润。

他们信雷斯吗。有人将信将疑，有人心里直打鼓。一个前科犯拿出这么惊天的项目，换谁都得犯嘀咕。可那份合同做得实在太逼真，印章多得晃眼，那笔买卖画的饼又实在太大。将信将疑，终究抵不过白花花的分成。

人心就是这样。怀疑是有的，可只要好处够大，怀疑就会自己找个台阶下来。一个接一个，几个帮手都上了船。上了船，就都成了这条船上的人。到这一步，散伙比继续更可怕。

上了船，口径就得统一。对外，他们众口一词。这是葡萄牙银行的绝密项目，为安哥拉注资，半个字不能声张。链条上的每一个人，都被这套说辞和满纸印章说服，也都被那笔分成拴住。谁要较真去核实，就是不识大体，就是给国家的大人物添乱。就这样，唯一能拆穿骗局的那道核实手续，被「保密」两个字牢牢焊死了。

回头看，雷斯站在这套精巧的说辞背后，几乎什么力气都没费。他没有撬锁，没有威胁谁，也没闯进任何一间戒备森严的库房。他更没碰过一台机器、一张钞纸。他只是造了一张纸，编了一个合情合理的故事。剩下的，是制度自己的规矩，替他把一道道该拦他的门反过来关严实。

一只皮箱，装着这份盖满印章的合同。拎着它的，是那位体面的荷兰掮客。火车和轮渡把他一程程送向北方，送向伦敦，送向那家给葡萄牙印了几十年钞票的老厂。他此行只有一个任务。他要说服这家最讲信誉的老厂，为一份彻头彻尾的假合同，开动那台真正的印钞机。
