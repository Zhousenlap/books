# 第五章：一幅假画，骗过了帝国元帅

先花两分钟认识一下这位买家，值得。

赫尔曼·戈林，纳粹德国帝国元帅，空军总司令，希特勒指定过的接班人，纽伦堡审判的头号被告——以及，欧洲历史上最疯狂的艺术囤积者之一。

这人对艺术品的胃口，用"贪婪"形容都嫌客气。战争期间，他借着第三帝国的权势，从被占领的欧洲各国搜刮艺术品：没收的、"购买"的、用犹太收藏家被抢的财产倒手换的……在他的乡间宫殿"卡琳庄园"里，名画堆得像仓库。他和希特勒（后者在筹建自己的"元首博物馆"）之间还存在某种收藏竞赛，两人的采购班子在欧洲艺术市场上明争暗抢。

而荷兰黄金时代的大师，正是这类藏家的心头肉。一幅维米尔？那是收藏皇冠上的钻石——希特勒手里就有维米尔（真的），戈林做梦都想要一幅。

1942年前后，机会来了。艺术市场的掮客给戈林的采购人牵线：有一幅新出土的维米尔，《基督与行淫的妇人》，可以谈。

交易的具体链条相当复杂，中间人好几道，我们不展开。值得记住的是成交方式：戈林付出的不是现金，而主要是**一批画**——用他手里137幅（这个数字来自战后清点，各资料略有出入）其他画作，换这一幅"维米尔"。整桩交易的估值高达一百多万荷兰盾，是战时艺术交易里数得着的大手笔。

顺便说个黑色幽默：后来有研究者核对戈林拿去换画的那批画，发现里面本身就混着赝品。用假画换假画，纳粹元帅和荷兰骗子在互相伤害，堪称艺术市场的黑暗森林。

戈林把这幅"维米尔"当宝贝收进卡琳庄园。据战后流传的说法，这是他最珍视的藏品之一——毕竟是他收藏里唯一的"维米尔"。

然后，1945年，第三帝国轰然倒塌。

盟军在奥地利阿尔陶塞的盐矿等地，起获了纳粹藏匿的海量掠夺艺术品。盟军里有个专门的部门，叫"纪念碑、艺术品和档案"小组（就是后来电影《盟军夺宝队》的原型），任务是把这些宝贝物归原主。清点战利品时，戈林收藏里那幅《基督与行淫的妇人》浮出水面。

荷兰方面一看，炸了：维米尔！我们的国宝维米尔在戈林手里！**谁卖的？**

顺藤摸瓜。交易链条一环环往回捋：掮客、银行账户、中间人……最后，线索指向了阿姆斯特丹绅士运河边的一栋豪宅，和它的主人，画家汉·凡·梅赫伦。

1945年5月底，荷兰刚解放没几天，凡·梅赫伦被捕。

罪名我们第一章说过：通敌，把国家文化财产卖给敌人。在当时的荷兰，这可不是普通的经济犯罪。全国正处在清算"对德合作者"的狂热期，几万人被抓，愤怒的人群在街上剃光"德国人的姘头"的头发。把国宝卖给戈林的人？往死里判都嫌轻。

凡·梅赫伦在拘押中扛了几天。他面临的选择，是人类骗子史上最奇特的一道二选一：

**选项A**：继续隐瞒。那幅画就是维米尔真迹，我就是把国宝卖给戈林的叛国犯。名声：爱国者眼中的败类。刑期：可能非常重，甚至有性命之忧。

**选项B**：说出真相。那幅画是我伪造的——连同博物馆里那幅被全国膜拜的《以马忤斯的晚餐》一起。名声：欺骗全国的伪造犯。刑期：诈骗和伪造，比叛国轻得多。但代价是：亲手砸碎自己"卖出过维米尔"的一切，并且把整个荷兰艺术界拖下水。

换句话说：**要么当叛国的英雄眼中钉，要么当欺诈的小丑**。

他选了小丑。据流传较广的版本，他在审讯中憋不住了，喊出的话大意是：你们以为我把维米尔卖给了戈林？蠢货，那画不是维米尔——是我画的！戈林买到的是赝品，被骗的是他！（原话措辞各书转述不一，标个待核实，但"主动供认伪造"这个核心情节是确凿的。）

审讯者的第一反应，前面说了：不信。

不信是有道理的。伪造犯给自己脱罪，编什么瞎话的都有；何况你说《以马忤斯的晚餐》是假的？那可是布雷迪乌斯认证、全欧洲专家背书、博物馆天价购入的杰作。你一个二流画家说是你画的，证据呢？

凡·梅赫伦的证据，就是他这双手。

于是，1945年夏天，在监视之下，一场史无前例的"表演"开场了：当局给他备好画室、画布、他要求的颜料和溶剂（据说还满足了他要酒和吗啡的要求，此节各书说法不一），几位见证人轮流围观。一个被控叛国的犯人，开始当众画他的第七幅——也是最后一幅——"维米尔"。

这幅画的题材，他选的是《圣殿中的年轻基督》（常译"少年基督与长老们"）。画了大约两个月。

据在场者回忆，看他画画的人，态度是肉眼可见地从怀疑变成瞠目结舌：那种十七世纪的光线、那种"维米尔味儿"，就在这个烟不离手的荷兰人笔下，一层一层浮现出来。

画还没画完，坊间风向已经变了。而接下来两年里发生的事，比画本身更精彩：科学登场了，舆论反转了，一场叛国案变成了一场全民围观的喜剧——凡·梅赫伦即将迎来他人生中最荒诞的一段时光：

**他这辈子最出名、最受欢迎的时刻，恰恰是他被全世界确认为骗子的时刻。**
