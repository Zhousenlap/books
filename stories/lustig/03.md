# 第三章：会"印钱"的魔法盒子

先说清楚，这一章讲的这台"印钞机"，我不会告诉你它具体怎么做——因为它压根就不能印钱，它是个骗人的道具。我要讲的，是它**为什么能骗到人**，好让你以后一眼认出这类把戏。这才是这本书该干的事。

这台机器，据说是一个做工精致的木头盒子，上面有几个旋钮和滚筒的开口，看起来像个正经的机械装置。勒斯蒂格会向他挑中的猎物——通常是船上某个贪心又自负的富人——神秘兮兮地展示它，说这是一台能复制钞票的机器。

然后他当场演示。他把一张真的百元美钞，从一头塞进去，煞有介事地转动旋钮，说这机器需要几个小时来"化学处理"。几个小时后，机器"吐"出两张百元钞票——一张是原来那张，另一张，崭新，逼真，拿到银行去居然真能兑开。

猎物的眼睛，此刻就直了。

你看，这个演示的杀伤力在哪儿？**在于它给了对方一个"眼见为实"的证据。** 猎物亲眼看着钞票进去，亲眼看着两张钞票出来，还亲手拿去验过是真的。人对自己的眼睛，有一种近乎迷信的信任——"我又不瞎，我可是亲眼看见的"。

可恰恰是这份自信，成了陷阱。因为他看见的一切，都是勒斯蒂格精心安排好让他看见的。那台机器里根本没有什么化学魔法，无非是事先藏好了真钞、用障眼法调了包——具体机关是什么不重要，重要的是：**你以为你在观察，其实你在被表演。** 魔术师的整个行当，就建立在这一点上。

演示成功后，剧本就顺理成章了：猎物两眼放光，急着要买下这台"下金蛋的鹅"。勒斯蒂格呢，偏偏还要欲擒故纵，装作万般不舍，说这是家传宝贝、说这东西太危险不能乱卖……越是不肯卖，对方越是志在必得，最后掏出一笔巨款——几千甚至上万美元，在那个年代是天文数字——买走了这个木头盒子。

而勒斯蒂格早就算好了下船的时间。等这位买家回家，兴冲冲地又塞进去一张钞票，等啊等，等来的只有一张原样吐回的旧钞。他再塞，再等，机器彻底"罢工"了。这时候他才明白过来：所谓几个小时的"化学处理时间"，根本就是为了给骗子留出**逃跑的时间差**。等他反应过来上当，那位伯爵早已远在千里之外。

这个"印钞机"骗局，藏着骗术的三味真火，我给你点破，你一辈子都能用它防身：

**第一，它专挑贪心的人下手。** 一个不相信天上掉馅饼的人，根本不会对"印钞机"动心。骗局的第一道筛子，筛掉的是老实人，留下的是贪心人。骗子最爱的猎物，永远是那个"想不劳而获"的人。所以防骗的第一课其实是防自己：**当一个机会好得不像真的，它就不是真的。**

**第二，它用一个真实的小甜头，钓一个巨大的假承诺。** 那第一张兑得开的真钞，是真的；但"这机器能一直印下去"是假的。骗子给你尝一口真的糖，让你相信后面一整罐都是真的。这个套路,后面卖铁塔、骗卡彭，会一而再再而三地出现。

**第三，它利用了"时间差"。** 骗局最怕的就是当场穿帮，所以高明的骗子总会给你一个"需要等待"的理由——等化学处理、等审批、等交割——本质上都是在为自己争取逃跑的窗口。**一切催着你现在就掏钱、又拦着你现在就验货的交易，都要亮红灯。**

靠着这套邮轮上的把戏，据说勒斯蒂格攒下了第一桶金，也攒下了名气。但小打小闹终究满足不了他。

在真正把目光投向岸上之前，据说他还在美国干过一票值得一提的买卖，对象是银行——一个远比船上富翁难对付的机构。这类故事里流传较广的一种说法是：他利用当时的金融漏洞和一批政府债券做文章，在一次交易里把银行和他自己都绕了进去，甚至一度被捕，却又凭三寸不烂之舌，说服银行相信"把他放了、别声张，对银行的名声更好"，最后不但脱身，据说还倒赚了一笔封口似的补偿（`待核实`：这段经历细节众多、版本不一，一手佐证稀薄，传奇成分很重）。

这个故事真假难辨，但它透露出勒斯蒂格骗术里一个反复出现的母题：**他总能让受害者相信，'不声张'对受害者自己更有利。** 无论是船上买了假机器不好意思声张的富翁，还是怕坏了名声的银行，勒斯蒂格都精准地利用了同一样东西——**人对'丢脸'的恐惧，常常大过对'损失'的心疼。** 这一手，他后来在巴黎会用到极致。

他把目光投向了岸上，投向了更大的猎物——银行、政府、乃至一整座城市。

而他这辈子最惊天动地的那一票，正在巴黎等着他。导火索，是1925年一张普通的报纸。
