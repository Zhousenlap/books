# 注释：史实、依据与争议

一本专门讲"别轻信骗子、别轻信一个太精彩的故事"的书，如果它自己在骗你，那就是天字第一号的讽刺。所以照老规矩，最后一章，我把正文里所有的史实、传说、存疑之处和艺术加工，一条条摊开给你看。

先声明一个特殊情况，而且这个声明对本书**格外重要**：本书是在一次**刻意不联网**的写作里完成的，正文所依据的，是我记忆中关于勒斯蒂格的通行说法与学界共识，而不是当场逐条重新核对的一手档案。凡涉及精确数字、精确日期、精确金额的地方，我一律标了 `待核实`，请你**千万不要**把本书任何一个精确数字当成已核实钉死的定论。

而勒斯蒂格这个题材，比本系列其他任何一本都更需要这句免责声明。原因下面第一条就讲。

## 一、一个根本性的警告：这个题材的史料天生就不牢靠

这是理解整本书的前提，请务必先读这一条。

**维克多·勒斯蒂格是一个职业撒谎者，他一生的核心技能就是伪造身份、编造经历、把假的说成真的。** 这就带来一个致命的问题：关于他的许多"事迹"，最原始的来源，往往就是他本人的自述、他向狱友或记者的吹嘘，以及后来的记者、传记作家在此基础上的转述和演绎。

这意味着：

- 他的很多传奇，**缺少独立的、第一手的档案佐证**（比如完整的庭审记录、受害者的正式报案、当时的一手新闻报道等相互印证）。
- 后世的通俗传记和文章，为了故事好看，普遍存在**添油加醋、细节层累**的现象——一个模糊的说法，经过几十年一遍遍转述，会被不断补上生动的对话、精确的数字和戏剧化的转折，越传越"清晰"，也越传越不可靠。
- 因此，本书讲的与其说是"勒斯蒂格的真实生平"，不如诚实地说是"**关于勒斯蒂格的、流传最广的那个版本**"。我尽力在讲述时区分了"较可靠"与"存疑"，但整体上，请你把这本书当成一个**传奇色彩很浓**的故事来读，而不是一份经过严格考据的史学定论。

一句话：**讲骗子的故事，本身就要提防被骗子（和讲述者）骗。** 这不是我推卸责任，这恰恰是本书主题的题中之义。

## 二、比较可靠的核心骨架

以下这些，是相对而言在多个来源里比较一致、可信度较高的部分（但仍非一手档案级别的铁证）：

- 历史上确有维克多·勒斯蒂格其人，是二十世纪上半叶活跃于欧美的著名骗子，以化名众多、骗术花样繁多著称。
- 他确实以"把埃菲尔铁塔当废铁卖给废品商"这一骗局闻名于世，这是他最著名的标签。
- 埃菲尔铁塔1889年为世界博览会而建，最初带有临时性质，早年确实在巴黎引发过审美争议、有文化名流联名反对；它后来因架设无线电／通讯天线等用途而得以保留。到1920年代，塔身老化、维护费用高，"是否该拆"在当时并非完全不可想象的话题——这为骗局提供了现实土壤。这一部分是本书里相对最扎实的背景。
- 他晚年确实卷入伪造美钞的案件，在美国被捕、定罪，并被关押于恶魔岛（Alcatraz）联邦监狱，最终死于狱中。
- 与他绑定的"骗徒十诫"（Ten Commandments for Con Men）是一份广为流传的文本，其内容核心确实是"如何取悦对方、博取信任、放低戒心"，而非编造谎言的技术。

## 三、存疑、多版本或传奇色彩浓厚的部分

这些是正文中我已经反复用"据说""据传""`待核实`"标注过的地方，在这里集中摊开：

1. **真名与出生。** "维克多·勒斯蒂格"很可能只是他众多化名之一；他的真实姓名、确切出生日期（常见说法为1890年1月4日）和出生地（常说是今捷克境内的霍斯廷 Hostinné，当年属奥匈帝国）都存在不同说法，且很大程度上源于他自己的交代，可信度打折。`待核实`。

2. **早年经历。** 少年时的语言天赋、会讲几国语言（有说三四种、有说五种）、"邮轮骗子"的生涯与那台"印钞机"（有称 Rumanian Box / 罗马尼亚盒子之类）——这些大多来自其自述与后世传记，细节难以独立证实，传奇成分高。`待核实`。

3. **卖铁塔骗局的诸多细节。** 这是全书传奇色彩最浓、也最缺一手佐证的部分：
   - 他伪装的确切官职（常说是"邮政/邮电部副总长"之类）、约见地点（常说是克利翁大酒店 Hôtel de Crillon）、邀请的商人数量、他报出的铁塔吨位（常说约7000吨）——各版本转述有出入。`待核实`。
   - 上钩的买家常被指名为**安德烈·普瓦松（André Poisson）**，以及"普瓦松因羞耻而不敢报案"的解释——这个说法流传极广、也很符合逻辑，但同样缺少独立的一手档案（如当年的报案或庭审记录）来铁证。这个名字和这段心理，很可能有后世加工的成分。`待核实`。
   - **"向普瓦松索贿以取信"**这一神来之笔，戏剧性极强，正因为太精彩，越要提防它是后人为了故事张力而强化甚至虚构的细节。`待核实`。
   - 他卷走的**具体金额**，从几万到几十万法郎的说法都有，无可靠一手账目。`待核实`。

4. **"卖了第二次"。** "第一次得手后又回巴黎故技重施"是勒斯蒂格招牌传奇的一部分，流传极广。但第二次是否真的走到了成交、有没有收到第二笔钱、他如何因买家报警而脱身——这些**一手证据极其稀缺**，是整个故事里最像"传奇加料"的一环。请把"卖了两次"当成一个精彩但**未经严格证实**的说法来听。`待核实`。

5. **骗艾尔·卡彭（Al Capone）。** "从卡彭手里拿走五万美元、原封不动归还、反获五千美元酬谢"的故事，几乎**完全依赖勒斯蒂格本人事后的讲述**，缺少来自卡彭一方或其他独立来源的印证。它太符合"骗子传奇"的戏剧模板，真实性存疑程度很高。当成一个漂亮的段子来读即可，别当铁的史实。`待核实`。

6. **伪钞案规模、越狱、卒年。** 假钞案的涉案金额与流通规模说法悬殊；"用床单越狱"的桥段常见于通俗叙述、具体经过难考；他常被记为**1947年**死于恶魔岛（一说死于肺炎），但确切卒年、死因、终年岁数以及"死亡证明职业栏写着'推销员'（salesman）"这个广为流传的细节，都属于**未经我当场核实**的说法。`待核实`。

## 四、艺术加工与措辞声明

- 正文里的各种**类比和心理点评**——"贪官比圣人更真实""大谎言更好使""保密是骗子的护身符""把'骗徒十诫'倒过来读就是反诈清单"等等——是我为了帮你理解骗术原理、提炼防骗要点而作的通俗提炼和评论，属于**叙事与说理**，不是史料原文，也不代表勒斯蒂格本人真有过这样清晰的自我总结。
- 书中凡涉及具体场景（酒店房间里的气氛、他翻报纸的神态、卡彭愣住的表情等），均为**基于流行叙述的场景化描写**，用于可读性，并非有一手记录的实况还原。
- 本书**未虚构任何具名人物**的真实身份，但正文中人物的具体对话、心理活动，多为依据流行版本所作的合理演绎，请勿当作逐字实录。
- 全书说书式的口语、反问、调侃，属于叙事风格；一切精确事实，以本注释的保留态度为准。

## 五、写作红线说明（重要）

本书是**教育性历史科普**，写作目的只有一个：通过讲述一个著名骗子的故事，帮助读者**看懂骗术背后的人性机关，从而学会识别与提防骗局**。

因此，本书对勒斯蒂格的种种骗术，**一律只作原理性、警示性的说明**，刻意**不提供任何可操作的行骗方法、步骤或话术模板**：

- 讲"印钞机"，我明确拒绝说明其机关，只讲它为什么能骗到人；
- 讲卖铁塔，我拆解的是"保密要求""制造紧迫""利用虚荣与贪心"这些**受害者该警惕的信号**，而不是教人如何设局；
- 讲"骗徒十诫"，我拒绝逐条罗列，只把它**反过来**当作反诈清单来用。

本书不美化犯罪。勒斯蒂格是一个给许多人造成真实伤害的罪犯，他的"传奇"底色是欺骗与背叛，结局是孤岛监狱里的潦倒死亡——这不是一个值得羡慕的人生，而是一个警世的标本。

## 六、依据与核实方式

**与严格考据的史学著作不同，本书未做联网核查，且题材本身（一个职业骗子的传奇）先天缺乏可靠一手史料。** 上文的骨架依据的是记忆中关于勒斯蒂格的通行叙述。请读者：

- 把本书当作一个**传奇色彩浓厚的普及读物**，而非考据定论；
- 对其中任何精确的姓名、日期、地点、金额、职务，以及"卖两次""骗卡彭"等最出彩的情节，都**保留一份怀疑**，若要引用，请自行查证严肃的传记与档案研究；
- 记住本书真正想留给你的，不是"勒斯蒂格具体在哪年骗了谁多少钱"这些易被加料的细节，而是那几条**永不过时的防骗常识**。

一本讲"别被骗子和好听的故事骗到"的书，最后这一课，必须先留给它自己：**来源不牢，就老实说来源不牢；靠不住的地方，就明明白白告诉你靠不住。** 以上。
